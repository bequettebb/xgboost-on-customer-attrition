사용한 데이터: https://www.kaggle.com/datasets/dhairyajeetsingh/ecommerce-customer-behavior-dataset

고객 이탈 예측을 해보기 위해 예시로 xgboost classifier를 사용.
결론은 (xgboost사용시) 파생변수 추가를 하나 안하나 고객센터 연결을 다수 했던 이용자들이 많이 이탈한 것으로 나타났다.
일부 수치 데이터들을 구간화해서 dummy변경 후 확인해봤으나 성능은 오히려 떨어져서 추가 작업은 패스.
결국 이 쇼핑몰은 고객센터에 문의/항의를 해야 할 정도의 문제가 있거나, 사소한 문제 발생시 고객센터에서 제대로 된 처리/응대를 하지 않은 것으로 보인다.
추가적으로 고객센터에 접수/처리된 문의 유형과 내용,결과를 우선 파악한 후에 쇼핑몰의 서비스나 운영방침을 변경해야 한다.
